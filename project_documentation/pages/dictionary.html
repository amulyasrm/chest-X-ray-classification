Technical Dictionary

Swin Transformer

A state-of-the-art hierarchical Vision Transformer (ViT) that uses shifted windows to compute self-attention. Unlike traditional Transformers that process the whole image globally, Swin processes images in local windows, making it computationally efficient while still capturing global context through hierarchical merging. This hierarchical structure allows it to detect diseases at multiple scales, from large shadows to small nodules.

Spatial Attention

A mechanism that allows the model to "focus" on specific geographical regions within an image. It highlights the most informative spatial locations (like the lung regions in a chest X-ray) while suppressing irrelevant background noise, effectively improving the signal-to-noise ratio in feature extraction. In our hybrid model, it acts as a filter that directs the CNN's attention to the most likely pathology sites.

CLAHE (Preprocessing)

Contrast Limited Adaptive Histogram Equalization is a sophisticated image preprocessing technique used to improve local contrast. Unlike global histogram equalization, CLAHE operates in small regions (tiles) to prevent the over-amplification of noise, ensuring that subtle details like nodules or pleural thickening are clearly visible to the AI, even in poor quality X-ray scans.

BCE Loss

Binary Cross Entropy is the gold standard loss function for multi-label classification. It measures the performance of a classification model whose output is a probability value between 0 and 1. In our project, it treats each of the 14 pathological findings as independent tasks, ensuring that learning one disease doesn't negatively interfere with the identification of another.

DenseNet121

A specialized CNN architecture where each layer is connected to every other layer in a dense block. This pattern addresses the vanishing-gradient problem and encourages feature reuse. Densenet is exceptionally well-suited for medical imaging because it extracts rich features while maintaining a relatively low parameter count, which helps prevent overfitting on clinical datasets.

Feature Fusion

The process of combining distinct types of information into a single representative vector. In our research, we fuse local features (textures/edges) from the CNN branch with global features (structural context) from the Swin branch. By concatenating these distinct "views", the model gains a holistic understanding of the patient's condition that neither branch could achieve alone.
